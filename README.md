# Proyecto: Análisis Exploratorio de Datos (EDA) sobre Adicción a Redes Sociales en Estudiantes

El repositorio contiene un analisis y visualizacion de datos los cuales se relacionan con las addcion a las redes sociales en estudiantes, utilizando una metodologia **CRISP-DM** para generar un enfoque estructurado.

## Bases de Datos Incluidas

1. **Students Social Media Addiction.csv**
   - Contiene información detallada sobre el uso de redes sociales, habitos de sueño, salud mental, rendimiento académico y estado sentimental de estudiantes de diferentes paises.
   - Variables clave: Edad, Genero, Nivel academico, Pais, Horas de uso diario, Plataforma mas usada, Afectacion academica, Horas de sueño, Puntaje de salud mental, Estado sentimental, Conflictos por redes sociales, Puntaje de adiccion.

2. **IMDB_5000_Movies.csv**
   - Base de datos de peliculas con informacion sobre directores, actores, generos, presupuesto, año, pais, idioma, y métricas de popularidad y puntuacion.
   - Util para ejercicios de análisis de datos, correlaciones y visualizacion de tendencias en la industria cinematografica.

3. **Estudiantes_programas_academicos_y_Extension_20260128.csv**
   - Contiene informacion sobre estudiantes los cuales cursan un programa academico profesional con datos como: "AÑO","SEMESTRE","SEXO BIOLOGICO","PROGRAMA","PAIS NACIMIENTO","DEPARTAMENTO NACIMIENTO","MUNICIPIO NACIMIENTO","NUMERO MATERIAS INSCRITAS","NUMERO MATERIAS APROBADAS","NIVEL_ACADEMICO","ESTRATO"

## Metodología CRISP-DM aplicada

1. **Entendimiento del Negocio:**  
   Se define el contexto de la economia de la atencion y el impacto de las redes sociales en estudiantes.  
   Los objetivos son; identificar patrones de uso, evaluar impacto en bienestar, analizar riesgos y factores sociales.

2. **Entendimiento de los Datos:**  
   Se exploran las variables, se realiza limpieza y se describen las columnas principales de cada dataset.

3. **Preparación de los Datos:**  
   Limpieza de columnas, manejo de nulos y estandarizacion para análisis.

4. **Modelado y Visualización:**  
   Se generan graficos para visualizar dominancia de plataformas, relacion entre tiempo de uso y adiccion, impacto en rendimiento academico, influencia del estado sentimental, conflictos y salud mental, y correlaciones globales.

5. **Evaluación:**  
   Se invita al usuario a reflexionar sobre los insights mas criticos, plataformas de riesgo y el papel de la salud mental.

## Archivos principales

- `EDA_Students_Social_Media_Full_CRISPDM.ipynb`: Notebook generado automaticamente que documenta todo el proceso de análisis y visualización.
- `Students Social Media Addiction.csv`: Base de datos principal para el análisis de adicción a redes sociales.
